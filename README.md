# ROS_practice


#Instructions for Compiling viznav Software

## 1. Create workspace
Create a directory called **ROS_practice** where you normally put your source code (usually git)
```
        cd ROS_practice
        mkdir src
        cd src
        git clone git@github.com:leochien1110/ROS_practice.git
```

## 4. Build workspace
```
        cd ../..
        catkin_make
```
